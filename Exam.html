CDN

CDN se conoce como la red de distribución de contenidos, es una red de computadoras que tienen copias de los datos en donde su función es darle una mayor capacidad a la banda para que los usuarios puedan entrar a los datos de la red. Asi los usuarios tienen acceso mas rapido a los datos, pues estan mas serca de ellos.

HISTORIA

- HTML -

HTML inicia en 1980, creado por el físico Tim Berners-Lee, al proponer un sistema de "hipertexto" para Internet en conjunto con el ingeniero Robert Cailliau, y su propuesta fue conocida como WorldWideWeb (W3). El primer documento formal en HTML fue publicado en 1991. En 1993 se solicitó para que se convertiera a estándar, ya que se habian definido varias etiquetas, como imágenes, tablas y formularios, pero no se consiguio. Hasta 1995 se consiguio el estándar HTML 2.0. 1997 Incorporan los applets de Java y el texto alrtededor de imagenes. 1998 HTML 4.0 con una nueva versión mejorada, incorporando CSS y sus hojas de esilo, scripts, tablas complejas y mejores formularios. 1999 se publicó HTML 4.01 sin grandes cambios. 2008 Se comienza a trabajr en HTML 5, por una nueva organización creada por Apple, Mozilla y Opera.







- CSS -

CSS(CascadingSyle Sheets) Hojas de estilo, mecanismo para aplicar diversos formatos a los documentos. Propuesta lanzada entre 1994 y 1995 por Wium Lie y Bert Bos. En 1995 se comenzo con el desarrollo y estandarización de este. 1996 Se publicón la primera recomendación oficial.1998 se publicó la segunda recomendación oficial, nivel 2. En 2009 Se realizo la ultima actualización. CSS nivel 3 comenzo a desarrollarse en 1998 pero aún no se ha lanzado de forma oficial
